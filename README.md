# Manim Animations

Welcome! This repository contains the code for my [Manim](https://www.manim.community/) animations featured on my YouTube channel: [KushDoesSmth](https://www.youtube.com/channel/UC5g58qKxuQ7IfIc_XJKrpCw).

## About

Here you'll find Python scripts and scene files used to create mathematical and educational animations with Manim.

## Getting Started

1. **Clone the repository and make a virtual enviorment:**

   ```bash
   git clone https://github.com/yourusername/anims.git
   cd anims
   python -m venv venv
   ```

   - Source the venv

2. **Install Dependencies:**

   ```bash
   pip install -r requirements.txt
   ```

3. **Render an animation:**
   ```bash
   manim file.py FileScene
   ```
